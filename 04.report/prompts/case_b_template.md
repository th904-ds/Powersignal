## Case B: 미래 날짜 예측 리포트

지금 이 리포트는 사용자가 선택한 미래 날짜(오늘 기준 +1~+7일) 기준으로 작성한다. 아래 5개 항목의 내용을 이 순서대로, 하나의 이어지는 본문에 녹여서 작성한다.

**모든 문장에 "예상", "전망", "예측" 중 하나의 표현을 반드시 사용한다.** 확정된 사실처럼 서술하지 않는다 (예: "SMP가 158원이다" 대신 "SMP가 158원 수준으로 예상된다").

1. 선택 날짜 SMP 전망 — 그날 예측 SMP 수준과 고가/저가가 예상되는 시간대를 언급한다.
2. 기상 기반 수요 전망 — 기온 예측치를 근거로 냉난방 수요에 미칠 영향을 전망한다.
3. 업종별 전력비 부담 전망 — 사용자가 속한 업종 기준으로 그날 예상되는 전력비 부담(부담 지수·등급)을 전망한다.
4. 경제성 DR 참여 권고 — 그날 예상 DR 스코어와 추천 시간대, 예상 수익을 근거로 참여를 권고한다.
5. 신뢰성 DR 발령 가능성 전망 — 예비력 예측 데이터를 근거로 그날 신뢰성 DR이 발령될 가능성이 있는 시간대를 전망한다.
